Project 6 — Textured Model Viewer

Overview

The purpose of Project 6 is to practice per-fragment shading and texturing.

Specifics

  • Create a new remote repository called "project6" on classes.csc.lsu.edu, and clone it to your local computer.

  • Add the four WebGL utility libraries and copy your HTML and JavaScript sources from Project 5.

    • webgl-utils.js
    • webgl-debug.js
    • cuon-utils.js
    • cuon-matrix.js
    • Viewer.js
    • Viewer.html

  • This application will take tables of vertices, normals, texture coordinates, and triangles in JSON format. Render this data using per-fragment lighting and texturing, following the template of “Per-fragment Illumination” in the notes. Allow the user to tumble the shaded model using the mouse.
