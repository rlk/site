Project 1 — Getting Started

Overview

The purpose of Project 1 is to ensure that you can organize HTML and JavaScript code into a working WebGL project and commit that working project to a Git repository.

The code in question will come right out of Chapter 2. At this time, we'll have only a light understanding of how this code truly works. We'll come to understand it more deeply over the next couple of weeks.

Specifics

  • Read chapter 2 of the WebGL Programming Guide, at least through page 40. This chapter gives a high level overview of almost every aspect of 3D programming for the web, including the necessary HTML, a bit of JavaScript with WebGL, and an early taste of the GPU programming language. It includes a very simple example that draws a single WebGL point to an HTML canvas.

  • Following the instructions given in Using Git with classes.csc.lsu.edu, create a new remote repository called "project1", and clone it to your local computer.

  • Download the following JavaScript libraries and add them to your project's working directory:

    • webgl-utils.js
    • webgl-debug.js
    • cuon-utils.js

  • Type in the HelloPoint1 example. This will include:

    • HelloPoint1.html from Listing 2.4.
    • HelloPoint1.js from Listing 2.5.

  • There are two corrections to be made to this code:

    1. You'll note that the script source paths in HelloPoint1.html point to a different directory. These paths are incorrect for your project, as the JavaScript files that they mention are in the project's working directory. Remove the "../libs/" portion of each path to correct them.
    2. The background is black and the point is red. This being LSU, everything must be purple and gold. Change the background to purple and the point to gold. See Chapter 2 to determine how.

  • Confirm that the code works by loading the HTML file in Chrome. It should look something like Figure 2.9.

  • Referring again to our Git instructions, add the five files you've created to the working copy and commit them to the local repository.

  • Push these changes from your local repository to your remote repository on classes.csc.lsu.edu before the deadline.
