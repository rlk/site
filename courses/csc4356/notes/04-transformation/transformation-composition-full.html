Transform Composition

We can thus define a composition all the way from an object's definition to the pixels of the screen.

$\V{v}_o$  Object coordinates
$\V{v}_w = \M{M}\cdot\V{v}_o$  World coordinates
$\V{v}_e = \M{V}\cdot\V{v}_w$  Eye coordinates
$\V{v}_c = \M{P}\cdot\V{v}_e$  Clip coordinates
$\V{v}_d = \V{v}_c\,/\,v_{cw}$ Normalized device coordinates
$\V{v}_s = \M{S}\cdot\V{v}_d$  Screen coordinates
