CSC 4356 / ME 4573 — Interactive Computer Graphics

This courses introduces the theory and practice of 3D computer graphics using WebGL and the Javascript programming language.

3D computer graphics deals with geometry and illumination. Geometry describes the form of a scene and illumination determines how that scene appears.

Rendering is the process by which 3D geometry data and illumination calculations produce a 2D image. Your computer's Graphics Processing Unit (GPU) performs that process.
