HTML Elements

The only HTML element that is absolutely required is the canvas. This is where the rendering appears.

It's generally smart to add a helpful message to the text of the element. If all goes well, the user will never see this. But if the browser lacks WebGL support the user will be informed.

A few notes:

  1. The id of the canvas must coincide with the identifier requested during Initialization.
  2. The aspect ratio given during rendering should equal width / height.
  3. The interaction functions must be made properties of this canvas element.

This section may also include input elements permitting interactive manipulation of such variables as objectColor, lightColor, lightDirection, modelTranslateZ, and others.
