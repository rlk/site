CSC 4356 / ME 4573 — Interactive Computer Graphics

This course provides an introduction to 3D computer graphics, emphasizing real-time graphics programming using WebGL and JavaScript programming language. Topics include an introduction to JavaScript, the OpenGL ES shading language, the mathematics of 3D graphics, transformation, texturing, lighting, scene management, and interaction. Topics in software engineering, including project management and source code controle, are alse covered.

Students may expect to gain experience developing interactive 3D content running across Windows, Linux, or OSX in a web browser on their PC, laptop, or mobile device. No prior experience with 3D graphics is required.

Resources

  • Git

    Consistent use of version control is required. This is how software development is done in the real world.

    • Install a Git Client. Under OS X, you can use the version of Git included with Xcode. Under Linux, you can use the package provided by your distribution.
    • Pro Git (aka the Git Book) is the complete reference and tutorial.
    • Here are details on using Git with classes.csc.lsu.edu.
    • We use Git from the command line. Here is a gentle introduction to the shell if needed.
    • The GitHub tool does not suffice. We use Git, not GitHub.

  • Browser

    There are still incompatibilities among the major browsers that make portable software development challenging. To minimize the impact of this on the content of this course, everyone uses the same browser.

    • Install Google Chrome. Chrome runs under all major operating systems, so you can use your favorite device regardless of OS.
    • The Chrome Developer Tools are particularly useful for debugging and analysis.

  • Javascript

    Javascript is the native language of the web browser.

    • JavaScript: The Good Parts (not required) is an excellent introduction to the language that emphasizes the useful aspects of the language while taking special care to avoid its pitfalls.
    • The Mozilla Developer Network maintains a complete set of JavaScript tutorials and references.

  • WebGL

    WebGL is the JavaScript API used to communicate with and control the GPU.

    • The Khronos Group maintains the WebGL specification and reference.
    • The OpenGL ES Shading Language is used to program the GPU.
    • Google maintains a list of WebGL demos if you'd like some inspiration.

  • HTML & CSS

    While not the focus of the course, we'll make some use of HTML and CSS, the languages that define the structure and appearance of web content.

    • The Mozilla Developer Network maintains documentation and tutorials on both HTML and CSS.
    • W3Schools hosts similarly comprehensive HTML and CSS documentation if you seek further references.

  • Text Editor

    Get a decent programmer's editor. Make sure it has syntax highlighting for Javascript and HTML. There are many excellent free code editors for all OSs, several of which run cross-platform.

    • Many Windows users like Notepad++.
    • I use Sublime Text, but it's not free.
    • Vim and Emacs have remained popular for decades, and for good reason.
    • Do not use Notepad or Wordpad.
