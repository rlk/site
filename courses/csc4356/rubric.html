CSC 4356 / ME 4573 — Interactive Computer Graphics

Programming Project Scoring Rubric

Criteria Inadequate Needs Improvement Meets Expectations Exceeds Expectations
Syntax
  • Indentation distance is inconsistent.
  • Lengthy regions of blank lines.
  • No explanatory comments.
  • Indentation has a mixture of tabs and spaces.
  • No explanatory comments except author's name.
  • Indentation distance is consistent.
  • Indentation is either all tabs or all spaces.
  • Each function or object is documented as to its purpose.
  • In-line commentary elucidates complex code.
Structure
  • Significant quantities of dead or useless code.
  • Redundant code.
  • Heavy use of global variables.
  • Use of member variables in lieu of argument passing.
  • Significant quantities of commented-out code.
  • Lengthy functions that do many tasks.
  • Each function performs one well-defined task.
Resource Management
  • Tasks performed in the wrong place (such as initialization tasks performed in the render loop) resulting in resource leaks.
  • Repeated resource acquisition without release.
  • Tasks performed in the wrong place (such as initialization tasks performed in the render loop) resulting in inefficiency.
  • Unnecessary repetition of resource acquisition and release.
  • Application is maximally efficient and allows resources to release upon exit.
  • All acquired resources released prior to exit.
