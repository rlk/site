Project 4 — Better Function Grapher

Overview

The purpose of Project 4 is to exercise the use of transformation for viewing. You will modify your implementation of Project 3 to improve the appearance of the renderered function and to allow the user to interactively view it from any angle and distance.

Specifics

  • Create a new remote repository called "project4" on classes.csc.lsu.edu, and clone it to your local computer.

  • Copy the five files from your project3 directory into your project4 directory.

    • Grapher.html
    • Grapher.js
    • webgl-utils.js
    • webgl-debug.js
    • cuon-utils.js

    You'll also need the textbook's matrix library.

    • cuon-matrix.js

    As always, format your code consistently and remove any dead code.

  • Display the 1-x2-z2 mesh as follows:

    • Render the surface of the mesh with colored triangles, as before.
    • Also render a grid of black lines on top of the triangle mesh.
      (This will probably involve the introduction of a uniform variable to disable the appearance of color, like the Light uniform used by this code for this demo.)

  • Break the code into separate initialization and draw functions, and allow interaction as follows:

    • Click and drag in the canvas to rotate the mesh relatively.

      • Click and drag vertically to rotate the mesh about the X axis.
      • Click and drag horizontally to rotate the mesh about the Y axis.

    • Provide an HTML range slider to move the mesh forward and back along the Z axis.

  • Use transformation matrices where needed:

    • Use a perspective projection matrix to produce the 3D effect.
    • Use model rotation and translation matrices to produce the interaction.

  • Add the files you've created to the project's working copy and commit them to the local repository. Push these changes from your local repository to your remote repository on classes.csc.lsu.edu before the deadline.
