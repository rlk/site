CSC 4356 / ME 4573 — Interactive Computer Graphics

This course provides an introduction to 3D computer graphics, emphasizing real-time graphics programming using OpenGL and the C or C++ programming language. Topics include the fixed function and programmable 3D pipelines, transformation, interaction, texturing, lighting, performance analysis, optimization, and a variety of intermediate and advanced topics.

Prerequisites include experience with the C or C++ programming language (CSC 1253-1254 or equivalent) and working understanding of data structures (CSC 3102 or equivalent). No prior experience with 3D graphics or OpenGL is required.

Assignments

Assignments include three individual programming projects with informal in-class demonstrations plus one final programming project with a formal in-class presentation. This work may be done using the hardware and OS preferred by the student. Sufficient hardware is also made available by the instructor as needed. Graduate students will have one additional writing assignment associated with the final.

  • Project 1: Basic Interactive Rendering
  • Project 2: The Programmable Pipeline
  • Project 3: Rendering to Textures
  • Final Project

The grading of each assignment has four components, all of which must be completed to receive an A grade:

Implementation

The successful implementation of program code completing the requirements described by the project specification. This should be packaged in a ZIP with all source code and assets necessary to reproduce the project. Executable binaries are not necessary, and should be omitted to minimize download size.

Documentation

The creation of a web site giving the implementation ZIP, at least three screen shots of the project running, and a description of what you did and how you did it. Convince the instructor that you understand what you've done. This must be a public site. It will be linked from the course web site, and your classmates will see it.

Demonstration

The live, in-class demonstration and description of your completed project. Be ready for this. Be certain that your project runs correctly on whatever computer you intend to use. Test before class. Programming project demonstrations will be informal, but the final project demonstration will be formal, with a prepared presentation.

Lagniappe

Something extra. You are free to enhance your submission in any way you like. Your addition should be significant, creative, and should distinguish your implementation from those of your classmates.

To submit an assignment, simply email the URL of your documentation web site to the instructor. Do so before class on the scheduled due date. To be fair to everyone in the class, we can't allow late submissions. The scheduled due date stands, regardless of whether your in-class demonstration falls on that date.

Useful Code

It's difficult to do anything interesting in real-time computer graphics without taking advantage of existing sources and libraries. While most useful libraries are included by default or otherwise easy to install under Linux and Mac OS X, resolving cross-platform dependencies is difficult under Windows. For this reason, I provide the Windows PITA, which includes a number of dependencies useful for 3D computer graphics.

Windows-PITA-2010.zip — Installation in Visual Studio
Windows-PITA-2011.zip — Installation in MinGW

In addition to these binaries, the following sources may be of use.

• hello-world.c — Makefile

A very simple example program using OPENGL and GLUT to display a teapot.

• Util3D

A set of OPENGL code modules providing services that may be useful in creating complex renderings. Several of these modules are already listed on the Code page, but a few more simple examples appear here.
